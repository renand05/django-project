## Getting Started

To start the project it is needed to have docker installed.


### Installation

1. Clone the repo
2. Run the repo using:
   ```sh
    docker-compose -up -build
   ```
3. Run the CLI using:
   ```sh
   python3 pokemon_etl_cli.py
  ```
